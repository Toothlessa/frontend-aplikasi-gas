| Layer         | Tanggung Jawab                               | Mirip di Laravel                    |
| ------------- | -------------------------------------------- | ----------------------------------- |
| `api/`        | Akses endpoint backend                       | Repository                          |
| `services/`   | Business logic, mapping data, error handling | Service                             |
| `store/`      | State management (data + action)             | Controller (dalam konteks frontend) |
| `components/` | UI (View)                                    | Blade / Vue View                    |
| `utils/`      | Helper, formatter, validator                 | Helpers / Traits                    |
| `types/`      | Definisi TypeScript (model)                  | Model / Resource                    |
